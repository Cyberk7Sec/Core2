● Task Manager
Ctrl+Shift+Esc · LIVE view
Processes
Think: PARTY LIST — who's at the party (running) and how drunk they are (CPU%). Kill crashers here.
Performance
Think: CAR DASHBOARD — gauges for CPU/RAM/Disk/GPU. Is the engine redlining?
Startup
Think: BOUNCER AT THE DOOR — decides who gets in when Windows starts. Disable slow ones to boot faster.
Services
Think: FACTORY MACHINES running in back. Start/stop background engines like Print Spooler or Windows Update.
Users
Think: HOTEL REGISTER — who is checked in and how much of the hotel's resources are they using.

■ MMC snap-ins (.msc)
Win+R → type command
eventvwr.msc
EVENT = EVIDENCE. Black box recorder of Windows. When it crashed, the log here is your crime scene photo. Look under Windows Logs.
diskmgmt.msc
DISK = DRAW A MAP. Visual editor for drive partitions. You see the drives as colored blocks — resize and create territories.
taskschd.msc
TASK SCHEDULER = PERSONAL ASSISTANT with a calendar. "Run this backup every night at 2am." Set it and forget it.
devmgmt.msc
DEV = DEVICES. Roster of every hardware plugged in. Yellow ⚠ = broken driver. Start here for hardware problems.
certmgr.msc
CERT = ID CARD CABINET. Stores the digital certificates browsers use to trust websites and identities.
lusrmgr.msc
LUS = LOCAL USERS. HR system for your PC — create accounts, assign groups, manage who exists locally.
perfmon.msc
PERF MON = SCIENTIFIC DATA LOGGER. Records counters over time. Deeper than Task Manager — saves reports.
gpedit.msc
GP = GROUP POLICY = THE RULEBOOK. Overrides individual settings. Enforce password length, block USB, restrict apps. Pro/Enterprise only.

◆ Extra .exe tools
Win+R → type command
msinfo32.exe
ms·INFO = SPEC SHEET. Just reads everything about your PC. CPU, RAM, BIOS, OS. Use when someone asks "what's your hardware?"
resmon.exe
RES·MON = RESOURCE + MONITOR with a MICROSCOPE. Like Task Manager but shows exactly WHICH file/port each process is touching.
msconfig.exe
ms·CONFIG = SAFE MODE LITE. Control what loads at boot without fully going into safe mode. Great for isolating startup problems.
cleanmgr.exe
CLEAN·MGR = ROOMBA. Vacuums temp files, recycle bin, old update files. Run "as admin" for deeper system file cleanup.
dfrgui.exe
D·FRG = DEFRAG = SORT THE BOOKSHELF. Reorganizes HDD data so the read head travels less. Remember: SSDs don't need this!
△
▲ Disk & registry
CMD as Admin · Handle with care
regedit.exe
REGISTRY = SOURCE CODE of Windows. Every setting lives here as key-value pairs. One wrong delete = broken OS. Always export a backup first!
chkdsk C: /f /r
CHK·DSK = HEALTH INSPECTOR for your drive. /f = fix errors, /r = find bad sectors. Needs a restart. Run when drive acts weird.
diskpart
DISK·PART = SCALPEL. Command-line partitioning, more powerful than the GUI. Used in scripts and WinPE recovery. list disk → select disk → clean.
sfc /scannow
SFC = SYSTEM FILE CHECK = the DOCTOR comparing your files to the originals. Repairs corrupted Windows files automatically. Always run as Admin.

⬡ Network CLI tools
CMD · Use in order when diagnosing no-internet
ipconfig
YOUR ADDRESS LABEL. First thing to check. /all for full details, /renew to refresh DHCP.
ping
SHOUTING "HELLO" — is the target even reachable? Try ping 8.8.8.8 first (no DNS needed).
nslookup
PHONE BOOK OPERATOR — is DNS resolving names to IPs? nslookup google.com
tracert
PACKAGE TRACKER — shows every router hop. Find where packets get dropped or slow.
netstat
WHO IS YOUR PC ON THE PHONE WITH? Lists all open connections and listening ports right now.
net use
PLUG REMOTE FOLDER IN as a local drive letter. net use Z: \\server\share
